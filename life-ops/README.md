# life-tatics

The standalone, modular time management and personal allocation engine for the produtividade workspace.

This submodule is completely decoupled from the overarching Algorithmic Life OS (`life/`) and its dependencies. It handles tracking routines, screentime mapping (dev work vs. resting), and timeblocks independently.

## Development

You can install this module locally:

```bash
cd life-tatics
poetry install
```

When changing the scope or features of this CLI, be sure to update `SPEC.md` and `.cursor/rules/submodule-life-tatics.mdc`.
